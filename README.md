# mini_compiler
# Mini Expression Compiler

A complete compiler pipeline built from scratch in **C++17** — featuring a hand-written lexer, recursive-descent parser, AST builder, stack-based bytecode code generator with backpatching, and a virtual machine executor.

Built as a systems programming project to explore how compilers translate high-level source code into executable instructions.

---

## Architecture

```
Source Code (.mc)
      │
      ▼
  ┌────────┐
  │  Lexer │  →  Tokenizes raw source into a stream of tokens
  └────────┘
      │
      ▼
  ┌────────┐
  │ Parser │  →  Recursive-descent parser builds an Abstract Syntax Tree
  └────────┘
      │
      ▼
  ┌─────┐
  │ AST │  →  Tree representation encoding operator precedence & structure
  └─────┘
      │
      ▼
  ┌─────────┐
  │ CodeGen │  →  Walks AST, emits stack-based bytecode with backpatching
  └─────────┘
      │
      ▼
  ┌────┐
  │ VM │  →  Stack machine executes bytecode instructions
  └────┘
```

## Build & Run

**Requirements:** g++ with C++17 support

```bash
# Clone the repo
git clone https://github.com/Owjasweeta/minicompiler.git
cd minicompiler

# Build
make

# Run a program
./minicompiler examples/example3.mc

# Show generated bytecode (IR)
./minicompiler examples/example3.mc --emit-ir

# Show Abstract Syntax Tree
./minicompiler examples/example3.mc --emit-ast

# Show both
./minicompiler examples/example3.mc --emit-ir --emit-ast
```

## Language Syntax

The compiler supports a simple imperative language with `.mc` extension.

```
// Variables & arithmetic
x = 3 + 4 * 2;        // operator precedence: x = 11
y = (3 + 4) * 2;      // parentheses: y = 14
print x;

// if / else
if (score > 90) {
    grade = 1;
} else {
    grade = 2;
}

// while loop
i = 1;
sum = 0;
while (i < 11) {
    sum = sum + i;
    i = i + 1;
}
print sum;             // prints 55
```

**Supported operators:** `+` `-` `*` `/` `<` `>` `==`  
**Control flow:** `if`, `else`, `while`  
**Statements:** assignment (`=`), `print`  
**Comments:** `// line comments`

## Key Concepts Implemented

| Concept | Where |
|---|---|
| Tokenization / Lexical Analysis | `lexer.cpp` |
| Recursive Descent Parsing | `parser.cpp` |
| Operator Precedence via Grammar | `parser.cpp` — `parseAdd` → `parseMul` → `parseUnary` |
| Abstract Syntax Tree | `ast.h` / `ast.cpp` |
| Stack-based IR / Bytecode | `codegen.h` |
| Backpatching (forward jumps) | `codegen.cpp` — `emit()` + `patch()` |
| Virtual Machine Execution | `vm.cpp` |
